# financial_agregator
Финансовый агрегатор для рынка Узбекистана

Описание:
Данный аргрегатор позволяет получить информацию об основных курсах валют в банках и финансовых организациях, о текущей стоимости золотых слитков и условиях кредитов, вкладов в банках Узбекистана.

Используемые технологии: 
ML для предсказания роста стоимости золотых слитков
